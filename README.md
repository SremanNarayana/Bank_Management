# Bank Management System in C++

## Description
This project is a console application developed in C++ for the purpose of managing a bank's customer transactions and account management services.

## Features
- **Account Creation:** Allows the creation of a new bank account.
- **Deposit Money:** Allows depositing money into an existing account.
- **Withdraw Money:** Allows withdrawing money from an existing account.
- **Balance Enquiry:** Allows checking the balance of an existing account.
- **Account Holder List:** Displays the details of all account holders.
- **Close Account:** Allows closing an existing account.
- **Modify Account:** Allows modifying details of an existing account.

## Getting Started

### Prerequisites
- A C++ compiler (like g++, clang++, or Visual Studio C++ compiler)
